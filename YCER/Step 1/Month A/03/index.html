🛡️ 영어 패턴 훈련소

패턴 1

It will be ready by [시점].

...까지 완료 가능 / 다 준비될 거야

📦 갈아 끼울 시점 부품 (명사 10개)
• this afternoon (오늘 오후)
• tomorrow morning (내일 아침)
• the end of this week (이번 주말)
• our next meeting (다음 미팅)
• noon (오늘 정오/낮 12시)
• 5 PM today (오늘 오후 5시)
• next Monday (다음주 월요일)
• the end of this month (이번 달 말)
• lunch (점심 시간 전까지)
• the deadline (마감 시한)
패턴 2

I need more time to [동사].

...하는 데 시간이 더 필요해

⚙️ 갈아 끼울 지연 사유 (동사 10개)
• check the details (디테일 확인)
• fix the errors (에러 수정)
• review the feedback (피드백 검토)
• confirm with the team (팀원 확인)
• prepare the slides (PPT 슬라이드 준비)
• translate the report (보고서 번역)
• test the system (시스템 테스트)
• double-check the figures (숫자 더블체크)
• get approval (승인 받기)
• update the data (데이터 업데이트)
패턴 3

Everything is [상태] right now.

지금 상황이 너무 ...해 / 다 ...된 상태야

🎨 갈아 끼울 현재 상태 (형용사/분사 10개)
• delayed (지연된 / 늦어지는)
• busy (바쁜 / 정신없는)
• stuck (완전히 꽉 막힌)
• messed up (꼬여서 엉망진창인)
• on hold (보류된 / 일시정지된)
• changing (계속 바뀌는 중인)
• complicated (복잡하게 꼬인)
• chaotic (전부 대 혼란인)
• overloaded (완전히 과부하 걸린)
• uncertain (모든 게 불확실한)
